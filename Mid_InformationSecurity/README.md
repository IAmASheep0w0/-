# 中项：信息安全工程师
> 参考视频教程：https://www.bilibili.com/video/BV1Dh411v7bf
## 中项考试（专业性）门槛
1. 低门槛
   - 电子商务设计师
   - 系统集成项目管理工程师
   - 信息系统管理工程师
2. 高门槛
   - 信息安全工程师
     - 新项目，题型不稳定
     - 案例题可能超纲（非教材内容）
## 考试形式
考试时间：**08:30~12:30**
1. 信息安全基础知识（机试，单选题）
   - 75个
   - 45/75合格
2. **信息安全应用实践（机试，案例题）**
   - 4~6题
   - （当前题型不是很固定）
   - 45/75合格
## 重点内容：密码学（试讲）
- 查漏补缺
  - 电子信封
    - 传输内容（长明文）用对称密钥加密，保证加密效率
    - 对称密钥用非对称密钥加密，保证密钥传输的安全性
